# 5W2H - Validação Automática de Relatório Final

## Aplicação de Gestão de Estágio

| Elemento | Descrição |
|----------|-----------|
| **What** (O quê?) | Implementar processo automático de validação de relatório final de estágio |
| **Why** (Por quê?) | Reduzir tempo de análise manual, minimizar erros e acelerar aprovação dos relatórios |
| **Who** (Quem?) | Desenvolvedores backend, coordenadores de estágio, estagiários |
| **Where** (Onde?) | Módulo de relatórios da aplicação web de gestão de estágio |
| **When** (Quando?) | Após submissão do relatório pelo estagiário |
| **How** (Como?) | Validar campos obrigatórios, verificar formatação, analisar conteúdo com regras configuráveis |
| **How Much** (Quanto custa?) | Estimativa: 40-60 horas de desenvolvimento + infraestrutura de processamento |
